# Entra Security Labs

> Real-world identity security. Deeper than docs, deeper than exam prep.

A public, community-focused lab resource for intermediate admins and security engineers working with Microsoft Entra ID. Every post covers a real pattern, real incident, or real misconfiguration — with hands-on steps you can follow in your own tenant.

**Live site:** https://mradelvand.github.io/entra-security-labs/

## Prerequisites

This repo uses a **hybrid strategy**: it does not teach foundational concepts from scratch. Each post links to [azurecertprep.github.io](https://azurecertprep.github.io) for the hands-on foundation, then goes deeper with real-world content.

If you're new to any topic covered here, complete the relevant azurecertprep challenge first.
